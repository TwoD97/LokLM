# NN - Titel des Tests

## Ziel

Kurz beschreiben, was mit diesem Test überprüft wird.

Beispiel:
Es wird geprüft, ob ein Benutzer sich erfolgreich registrieren kann.

## Vorbedingung

- Die Anwendung ist gebaut und gestartet.
- Die benötigten Dienste sind erreichbar.
- Der Tester befindet sich an der richtigen Ausgangsstelle.
- Benötigte Testdaten sind vorbereitet.

## Testdaten

| Feld | Wert |
|---|---|
| E-Mail | testuser@example.com |
| Benutzername | testuser01 |
| Passwort | Test1234! |
| Sonstiges | - |
