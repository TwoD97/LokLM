## Ziel

Es wird geprüft, ob der Benutzer eine Chat-Nachricht senden kann, eine Antwort erhält und die angezeigten Zitate/Quellen korrekt anklickbar sind.

## Vorbedingung

- Die Anwendung ist installiert und gestartet.
- Die Datenbank ist erreichbar, falls Chatverläufe gespeichert werden.
- Der Benutzer ist eingeloggt.
- Der Chatbereich ist erreichbar.
- Für die Testfrage sind passende Quellen oder Dokumente vorhanden.
- Internetverbindung ist vorhanden, falls externe Quellen verwendet werden.

| Feld             | Wert                                                                |
| ---------------- | ------------------------------------------------------------------- |

| Erwartete Quelle | Pflichtenheft / Testdokumentation                                   |
| Benutzerkonto    | testuser@example.com                                                |
| Besonderheit     | Antwort soll mindestens ein Zitat oder eine Quellenangabe enthalten |

## Schritte

1. Den Chatbereich öffnen.
2. Die Testfrage in das Eingabefeld schreiben.
3. Die Nachricht absenden.
4. Warten, bis die Antwort vollständig angezeigt wird.
5. Prüfen, ob die Antwort fachlich zur Frage passt.
6. Prüfen, ob mindestens ein Zitat oder eine Quellenangabe angezeigt wird.
7. Auf das Zitat oder die Quellenangabe klicken.
8. Prüfen, ob die passende Quelle geöffnet oder markiert wird.
9. Zurück zum Chat wechseln.
10. Prüfen, ob der Chat weiterhin bedienbar ist.

## Erwartet

1. Der Chatbereich wird ohne Fehlermeldung geöffnet.
2. Die Testfrage wird korrekt im Eingabefeld angezeigt.
3. Die Nachricht wird gesendet und im Chatverlauf angezeigt.
4. Eine Antwort wird vollständig geladen.
5. Die Antwort passt inhaltlich zur gestellten Frage.
6. Mindestens ein Zitat oder eine Quellenangabe ist sichtbar.
7. Das Zitat oder die Quellenangabe ist anklickbar.
8. Die passende Quelle wird geöffnet, angezeigt oder hervorgehoben.
9. Der Benutzer kann zum Chat zurückkehren.
10. Der Chat bleibt weiterhin funktionsfähig.
