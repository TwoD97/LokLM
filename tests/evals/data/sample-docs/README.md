# Sample-Docs

Platzhalter-Dokumente für die eval-pipeline. Jede `.txt`-Datei in diesem Ordner
wird vom `generate-dataset`-Script eingelesen, gechunkt und an den
Question-Generator gefüttert.

Diese drei Texte sind selbst geschrieben und dürfen ohne Lizenzbedenken im
Repo liegen. Sobald LokLM einen echten Zielkorpus bekommt (z.B. interne
Fach-PDFs, Lehrbücher), ersetzt man die Sample-Docs hier durch echte
Auszüge oder zeigt das `generate-dataset`-Script auf einen externen Ordner.

Was hier _nicht_ landen darf:

- Urheberrechtlich geschützte Texte ohne Genehmigung.
- Personenbezogene Daten.
- Texte mit denen LokLM später in der Produktion arbeitet (gehören in den
  echten Vault, nicht ins Repo).
